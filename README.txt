A simple and lightweight plugin to display the estimated reading time of your posts and pages.

== Description ==

WP Reading Time Estimator calculates the estimated reading time for any post or page based on the word count.  
It’s lightweight, accurate, and easily customizable.

**Features:**
* Automatically display reading time before/after content.
* Customize the text (e.g., "Estimated reading time: 5 minutes").
* Shortcode support: `[reading_time]`.
* Works with any theme.
* Translation ready.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-reading-time-estimator/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the ‘Plugins’ screen in WordPress.
3. By default, reading time will show at the beginning of your posts.
4. Use `[reading_time]` shortcode anywhere inside your content if you want manual placement.

== Frequently Asked Questions ==

= How is reading time calculated? =
It’s based on the average reading speed of 200 words per minute.

= Can I change the text? =
Yes! Go to **Settings > Reading Time Estimator** to change prefix/suffix text.
